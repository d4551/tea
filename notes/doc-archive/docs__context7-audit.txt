## 3) Workflow

```mermaid
flowchart TD
  subgraph TaskFlow["Task involves library"]
    A[Library needed]
  end
  A --> B{In llms-stack?}
  B -->|Yes| C[Use static docs first]
  B -->|No| D[Use Context7]
  A --> E[Need version-specific pattern]
  E --> D
  D --> F[resolve-library-id]
  F --> G[query-docs with specific question]
  G --> H[Apply examples to code]
```

When working with Pixi.js, Three.js, HuggingFace Transformers, Playwright, or Biome:

1. Use the Context7 MCP server (resolve-library-id, then query-docs).
2. Limit to 3 query-docs calls per question.
3. Pre-resolved IDs are in .cursor/rules/context7.mdc.

For Elysia, htmx, Prisma, Tailwind, DaisyUI: use llms-stack-refresh first. Use Context7 when you need version-specific or recent patterns.
